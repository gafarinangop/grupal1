Tablas en HTML5

Las tablas son muy útiles para organizar datos en un formato estructurado, con filas y columnas. Se crean utilizando la etiqueta <table>, con filas definidas por <tr> y celdas <td>.
Si por otro lado queremos estructurar la tabla con encabezado: La manera correcta luego del código "table" es colocar el código para encabezado: <thead> y, en caso de las columnas el código es <th>; para el cuerpo de la tabla luego de cerrar "thead" el código es <tbody>

Imágenes en HTML5

Las imágenes son un elemento fundamental en las páginas web, ya que ayudan a transmitir información visual y mejoran la experiencia del usuario. En HTML5, las imágenes se incorporan utilizando la etiqueta <img>.

Aquí te mostramos cómo usar la etiqueta <img> correctamente:

Sintaxis de la etiqueta <img>

La etiqueta <img> es una etiqueta vacía, lo que significa que no tiene una etiqueta de cierre. Sus atributos principales son:

  • src: Especifica la ruta de la imagen. Puede ser una URL absoluta o relativa.
  • alt: Proporciona un texto alternativo que se muestra si la imagen no se puede cargar. Este atributo es importante para la accesibilidad y para los motores de búsqueda.
  • width y height: Definen el ancho y alto de la imagen (en píxeles).

Ejemplos

Aquí hay algunos ejemplos de cómo usar imágenes en HTML:

1. Imagen básica

<img src="ruta/de/imagen.jpg" alt="Descripción de la imagen">

2. Imagen con tamaño específico

<img src="ruta/de/imagen.jpg" alt="Descripción de la imagen" width="300" height="200">

Consejos para usar imágenes de manera eficiente

  • **Optimiza tus imágenes**: Usa formatos de imagen adecuados (JPEG para fotos, PNG para gráficos transparentes, y SVG para imágenes vectoriales) y comprime las imágenes para mejorar el rendimiento de la página.
  • **Usa el atributo alt**: Siempre proporciona un texto alternativo que describa la imagen. Esto es crucial para usuarios con discapacidades visuales y para situaciones en las que la imagen no se carga.
  • **Aprovecha las imágenes responsivas**: Utiliza técnicas como el atributo srcset y sizes para que las imágenes se adapten a diferentes resoluciones de pantalla.

Imágenes responsivas

Con HTML5, puedes usar el atributo srcset para definir varias versiones de la misma imagen, asegurando que los usuarios obtengan la versión más adecuada según el tamaño de su dispositivo:

<img src="imagen-pequena.jpg" srcset="imagen-mediana.jpg 768w, imagen-grande.jpg 1200w" sizes="(max-width: 768px) 100vw, 50vw" alt="Imagen responsiva">

Hipervínculos en HTML5

Los hipervínculos son elementos clave en la navegación web, ya que permiten a los usuarios moverse de una página a otra o a diferentes secciones dentro de la misma página. En HTML5, los hipervínculos se crean con la etiqueta <a>.

Sintaxis de la etiqueta<a>

La etiqueta <a> requiere al menos el atributo href, que especifica la URL o la ubicación a la que se quiere enlazar. El texto o contenido dentro de la etiqueta <a> se convierte en un enlace clicable.

Ejemplos de uso

1. Hipervínculo básico

<a href="https://www.ejemplo.com">Visitar Ejemplo</a>

Este enlace lleva al usuario al sitio web "https://www.ejemplo.com".

2. Enlace a una sección en la misma página

Para enlazar a una sección específica de la misma página, se utiliza un identificador (ID) del elemento:

<a href="#imagenes">Ir a la sección de Imágenes</a>

El enlace llevará al usuario a la sección con el atributo id="imagenes".

3. Enlace a un archivo o descarga

<a href="documento.pdf" download>Descargar Documento</a>

El atributo download hace que el archivo se descargue automáticamente en lugar de abrirse en el navegador.

Atributos importantes de los hipervínculos

  • href: Define la URL a la que apunta el enlace. Puede ser una URL absoluta (un enlace externo) o relativa (un enlace interno).
  • target: Especifica dónde se abrirá el enlace. Los valores comunes son:
    • _self :Abre el enlace en la misma pestaña (comportamiento por defecto).
    • _blank : Abre el enlace en una nueva pestaña o ventana.
  • rel: Describe la relación entre el documento actual y el enlace. Es útil para la seguridad cuando se usa target="_blank" (por ejemplo, rel="noopener noreferrer").
  • title: Proporciona información adicional sobre el enlace que se muestra como un tooltip cuando el usuario pasa el ratón por encima.
